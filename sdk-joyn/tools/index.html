Tools

This section regroups a number of specific tools that can help you with your application development.

  • Standalone joyn service

    If your device is not natively joyn compliant, you should deploy a standalone joyn service on your device before using the joyn APIs from applications.

    You will find a step by step explanation on how to accomplish this in our dedicated guide.

  • Reference Implementation (RI)

    The Reference Implementation is an application with a very basic UI which allows you to test the joyn features (chat, file transfer, sharing) step by step. See also the source code of the RI here.
