Get to know joyn services via the RI application

The RI application will help you get a quick view of the different functionalities joyn APIs offer.

The RI application uses the joyn APIs and offers a basic UI to test the joyn features.

To install the RI, connect your device via USB. Make sure that USB debugging is active. Then use the following command:

adb install <SDK>\tools\joyn-RI.apk
