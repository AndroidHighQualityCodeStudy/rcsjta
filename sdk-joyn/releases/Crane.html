API version Crane

joyn Crane is based on RCS 5.2 specification.

Crane release introduces the following new features:
  • Automata capability which indicates if the remote is an automata or not.
  • Multimedia Session API which permits to create new multimedia services.

Note: Crane specification not yet published by GSMA.
