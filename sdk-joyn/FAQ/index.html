Frequently asked questions

Here we provide some answers to problems you may have encountered using joyn APIs

  • Why is my local configuration not taken into account by the provisioning application?

    Don’t forget to save the modification via the menu “Save”. This should be done folder by folder in the provisioning application.

  • How do I see SIP traces in the logcat ouput?

    By default the traces are not displayed in traces. To activate SIP traces, you should select the corresponding settings from the provisioning application (see folder “Logger”). There is the same kind of parameter for media traces (e.g. MSRP packet).
