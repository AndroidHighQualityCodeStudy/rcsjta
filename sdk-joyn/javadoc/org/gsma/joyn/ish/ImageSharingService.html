Class Overview

This class offers the main entry point to transfer image during a CS call. Several applications may connect/disconnect to the API. The parameter contact in the API supports the following formats: MSISDN in national or international format, SIP address, SIP-URI or Tel-URI.

public ImageSharing shareImage (String contact, String filename, ImageSharingListener listener)

Since: API Level 1.0

Shares an image with a contact. The parameter file contains the complete filename including the path of the image to be shared. An exception if thrown if there is no ongoing CS call. The parameter contact supports the following formats: MSISDN in national or international format, SIP address, SIP-URI or Tel-URI. If the format of the contact is not supported an exception is thrown.
