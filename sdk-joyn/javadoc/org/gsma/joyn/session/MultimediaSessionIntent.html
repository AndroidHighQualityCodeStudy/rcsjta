public static final String ACTION_NEW_INVITATION

Since: API Level Albatros

Broadcast action: a new multimedia session invitation has been received.

Intent includes the following extras:

  • EXTRA_CONTACT containing the MSISDN of the contact sending the invitation.
  • EXTRA_DISPLAY_NAME containing the display name of the contact sending the invitation (extracted from the SIP address).
  • EXTRA_SESSION_ID containing the unique ID of the multimedia session.
  • Service ID is read from the method Intent.getType() which returns the MIME type included in the intent and corresponding to the invoked service.

Constant Value: "org.gsma.joyn.session.action.NEW_SESSION"
