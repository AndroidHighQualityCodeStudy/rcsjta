Known Direct Subclasses
CapabilityService, ChatService, ContactsService, FileTransferService, ImageSharingService, MultimediaSessionService, VideoSharingService
CapabilityService Capability service offers the main entry point to read capabilities of remote contacts, to initiate capability discovery and to receive capabilities updates. 
ChatService Chat service offers the main entry point to initiate chat 1-1 ang group conversations with contacts. 
ContactsService Contacts service offers additional methods to manage RCS info in the local address book. 
FileTransferService This class offers the main entry point to transfer files and to receive files. 
ImageSharingService This class offers the main entry point to transfer image during a CS call. 
MultimediaSessionService This class offers the main entry point to initiate and manage new and existing multimedia sessions. 
VideoSharingService This class offers the main entry point to share live video during a CS call.
