Class Overview

Chat service offers the main entry point to initiate chat 1-1 ang group conversations with contacts. Several applications may connect/disconnect to the API. The parameter contact in the API supports the following formats: MSISDN in national or international format, SIP address, SIP-URI or Tel-URI.
