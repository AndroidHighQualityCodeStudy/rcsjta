public static final String ACTION_NEW_INVITATION

Since: API Level Blackbird

Broadcast action: a new IP call invitation has been received.

Intent includes the following extras:

  • EXTRA_CONTACT containing the MSISDN of the contact sending the invitation.
  • EXTRA_DISPLAY_NAME containing the display name of the contact sending the invitation (extracted from the SIP address).
  • EXTRA_CALL_ID containing the unique ID of the IP call.
  • EXTRA_AUDIO_ENCODING containing the audio encoding.
  • EXTRA_VIDEO_ENCODING containing the video encoding.
  • EXTRA_VIDEO_FORMAT containing the video format.

Constant Value: "org.gsma.joyn.ipcall.action.NEW_CALL"
