Class Overview

This class offers the main entry point to initiate and manage new and existing multimedia sessions. Several applications may connect/disconnect to the API.

public MultimediaSession initiateSession (String serviceId, String contact, String sdp, MultimediaSessionListener listener)

Since: API Level Albatros

Initiates a new multimedia session with a remote contact and for a given service. The SDP (Session Description Protocol) parameter is used to describe the supported media. The parameter contact supports the following formats: MSISDN in national or international format, SIP address, SIP-URI or Tel-URI. If the format of the contact is not supported an exception is thrown.

Parameters
serviceId Service ID
contact Contact
sdp Local SDP
listener Multimedia session event listener

public boolean sendMessage (String serviceId, String contact, String content, String contentType)

Since: API Level Albatros

Sends an instant message to a contact and for a given service. The message may be any type of content. The parameter contact supports the following formats: MSISDN in national or international format, SIP address, SIP-URI or Tel-URI. If the format of the contact is not supported an exception is thrown.
