Class Overview

IP call player offers an interface to manage the IP call player instance independently of the joyn service. The IP call player is implemented in the application side. The IP call player captures the audio/video from the device micro/camera, encodes the audio/video into the selected formats, streams the encoded audio samples and video frames over the network in RTP.
